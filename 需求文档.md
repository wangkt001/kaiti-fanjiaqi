# "豫见好物"河南文创产品销售平台需求文档

## 1. 项目背景与目标

### 1.1 项目背景

河南作为中华文明的重要发祥地，拥有丰富的历史文化资源和独特的地域文化特色。然而，当前河南文创产品销售存在以下痛点：

- **销售渠道有限**：传统销售模式依赖线下门店和展会，覆盖范围有限
- **文化传播不足**：消费者难以深入了解文创产品背后的文化内涵
- **供需信息不对称**：卖家无法精准触达目标客户，买家难以找到心仪的文创产品
- **文化 IP 价值未充分转化**：丰富的文化资源未能有效转化为商业价值

随着数字化经济的发展，亟需一个垂直化的文创产品销售平台，以促进河南文创产业的高质量发展。

### 1.2 项目目标

- 搭建专业化的河南文创产品销售平台，实现文化资源与商业价值的有效转化
- 强化文创卖家、买家与文化资源之间的联结，构建文化生态圈
- 提供深度文化内容展示，让用户了解文创产品背后的文化故事
- 探索"文化 + 电商"的创新发展模式，推动河南传统文化的创新性发展
- 形成可复用的文创电商平台开发范式，为其他地区文创产业发展提供参考

## 2. 用户角色

| 角色   | 注册方式            | 核心权限                                                                 |
| ------ | ------------------- | ------------------------------------------------------------------------ |
| 买家   | 手机号/邮箱注册     | 浏览商品、了解文化背景、搜索筛选、下单购买、评价反馈、文化互动、收藏点赞 |
| 卖家   | 资质审核 + 手机注册 | 上传商品信息、设置文化标签、管理订单库存、查看销售数据、优化推广策略     |
| 管理员 | 系统分配            | 用户管理、商品审核、文化内容合规校验、系统维护、运营监控                 |

### 2.1 角色详细说明

#### 2.1.1 买家

- **身份特征**：文创爱好者、收藏者、游客、文化研究者
- **核心需求**：
  - 浏览和购买河南特色文创产品
  - 了解产品背后的文化故事和历史渊源
  - 参与文化互动，与其他爱好者交流
  - 收藏喜欢的商品和资讯
- **使用场景**：
  - 通过分类或搜索找到心仪的文创产品
  - 查看商品详情，了解文化背景和工艺说明
  - 下单购买，支持多种支付方式
  - 对商品进行评价，分享使用体验
  - 浏览文化资讯，了解河南文化动态

#### 2.1.2 卖家

- **身份特征**：文创企业、手工艺人、博物馆、文化机构
- **核心需求**：
  - 高效上传和管理商品信息
  - 精准设置文化标签，触达目标客户
  - 管理订单和库存，及时处理发货
  - 查看销售数据和用户画像，优化经营策略
- **使用场景**：
  - 提交卖家资质认证，通过审核后开店
  - 发布商品，填写详细的文化信息（文化背景、工艺说明、产地等）
  - 设置文化标签（如：豫剧、少林、牡丹、青铜器等）
  - 处理订单，更新物流信息
  - 查看销售报表，分析用户偏好
  - 回复用户评价，维护店铺形象

#### 2.1.3 管理员

- **身份特征**：平台运营人员、文化审核专家
- **核心需求**：
  - 管理用户账号，审核卖家资质
  - 审核商品信息和内容合规性
  - 发布文化资讯，运营平台内容
  - 监控系统运行，保障交易安全
- **使用场景**：
  - 审核卖家提交的资质材料
  - 审核商品上架申请，确保文化信息准确性
  - 发布文化资讯、非遗知识、展览活动等信息
  - 处理用户举报和纠纷
  - 查看平台运营数据，制定运营策略

## 3. 功能需求

### 3.1 用户管理模块

#### 3.1.1 用户注册

- 支持手机号注册和邮箱注册
- 买家注册：填写基本信息（昵称、手机号、密码）
- 卖家注册：提交资质认证（营业执照、身份证、作品集等）
- 实名认证：卖家需完成实名认证才能发布商品

#### 3.1.2 用户登录

- 支持账号密码登录
- 支持手机验证码登录
- 支持第三方登录（微信、QQ）
- 使用 JWT Token 认证

#### 3.1.3 个人信息管理

- 查看和修改个人基本信息
- 上传和修改头像
- 修改联系方式（手机号、邮箱）
- 修改登录密码
- 买家：管理收货地址
- 卖家：管理店铺信息（店铺名称、logo、简介）

#### 3.1.4 用户注销

- 支持账号注销
- 注销前需完成订单清算和资金结算

### 3.2 商品管理模块

#### 3.2.1 商品发布（卖家）

- 填写商品基本信息：
  - 商品名称、描述、价格、库存
  - 商品分类（传统工艺品、文化创意产品、非遗传承、博物馆文创、地方特色）
  - 商品图片（主图、详情图，支持轮播）
- 填写文化信息（必填）：
  - 文化背景：产品背后的历史故事、文化渊源
  - 工艺说明：制作工艺、传承技艺
  - 设计师/传承人信息
  - 产地信息（洛阳、开封、安阳等）
  - 材质说明
  - 文化 IP 归属（如：河南博物院、少林寺等）
- 设置文化标签：
  - 从预设标签库选择（豫剧、少林、牡丹、青铜器、陶瓷、刺绣等）
  - 支持自定义标签（需审核）
- 设置 SKU 规格（可选）：
  - 颜色、尺寸、款式等
  - 不同 SKU 可设置不同价格和库存
- 设置物流信息：
  - 运费模板
  - 是否支持礼品包装
  - 是否支持定制

#### 3.2.2 商品编辑（卖家）

- 修改已发布商品的信息
- 修改商品价格和库存
- 上下架商品
- 文化信息修改需重新审核

#### 3.2.3 商品删除（卖家）

- 删除已发布的商品
- 有订单的商品不可删除

#### 3.2.4 商品审核（管理员）

- 审核新发布商品
- 审核商品文化信息的准确性
- 审核不通过时给出修改意见
- 支持批量审核

#### 3.2.5 商品列表

- 按分类展示商品
- 支持多种筛选条件：
  - 文化标签筛选
  - 产地筛选
  - 价格区间筛选
  - 工艺类型筛选
  - 文化 IP 筛选
- 支持多种排序方式：
  - 综合排序
  - 销量排序
  - 价格排序（升序/降序）
  - 发布时间排序
  - 热度排序
- 支持搜索功能：
  - 关键词搜索
  - 智能推荐（基于用户浏览和购买历史）

#### 3.2.6 商品详情

- 展示商品基本信息
- 展示商品图片（轮播图）
- 展示文化信息专区：
  - 文化背景故事
  - 工艺流程说明
  - 设计师/传承人介绍
  - 产地文化介绍
  - 材质说明
  - 文化 IP 介绍
- 展示文化标签
- 展示 SKU 规格选择
- 展示用户评价
- 展示相关推荐商品
- 购买按钮（加入购物车、立即购买）
- 收藏按钮、分享按钮

### 3.3 交易管理模块

#### 3.3.1 购物车

- 添加商品到购物车
- 修改购物车商品数量
- 删除购物车商品
- 批量结算

#### 3.3.2 商品购买

- 选择商品规格和数量
- 选择收货地址
- 选择配送方式
- 选择支付方式
- 使用优惠券（如有）
- 填写礼品留言（如选择礼品包装）
- 提交订单

#### 3.3.3 订单管理

- 查看订单列表：
  - 全部订单
  - 待付款
  - 待发货
  - 待收货
  - 已完成
  - 已取消
- 查看订单详情：
  - 商品信息
  - 收货信息
  - 支付信息
  - 物流信息
- 订单操作：
  - 取消订单（未发货）
  - 确认收货
  - 申请售后
  - 删除订单

#### 3.3.4 支付功能

- 集成支付宝支付
- 集成微信支付
- 支持银行卡支付
- 支付状态回调处理

#### 3.3.5 物流跟踪

- 对接快递鸟/快递 100 API
- 实时查询物流信息
- 物流状态更新通知

### 3.4 评价管理模块

#### 3.4.1 商品评价

- 已完成订单可评价
- 评分（1-5 星）
- 文字评价
- 上传图片（最多 9 张）
- 匿名评价选项
- 评价提交后不可修改

#### 3.4.2 评价回复

- 卖家可回复用户评价
- 管理员可回复重要评价
- 支持多级回复（楼中楼）

#### 3.4.3 评价点赞

- 用户可对评价进行点赞
- 显示点赞数量
- 点赞后可取消

#### 3.4.4 评价查看

- 商品详情页显示所有评价
- 支持筛选（有图评价、好评、中评、差评）
- 支持排序（时间排序、点赞数排序）

#### 3.4.5 评价统计

- 显示商品总体评分
- 显示评分分布（5 星、4 星、3 星、2 星、1 星占比）
- 显示评价标签（如：质量好、物流快、包装精美等）

## 4. 非功能需求

### 4.1 性能需求

- **响应时间**：
  - 页面加载时间不超过 2 秒
  - API 接口响应时间不超过 500ms
  - 图片加载时间不超过 1 秒
- **并发能力**：
  - 支持同时在线用户数不低于 1000 人
  - 支持每秒 100 次以上并发请求
- **数据库性能**：
  - 复杂查询响应时间不超过 1 秒
  - 支持百万级数据量

### 4.2 安全需求

- **认证安全**：
  - 使用 JWT Token 认证
  - 密码加密存储（BCrypt）
  - 支持手机验证码登录
  - 支持第三方登录
- **数据安全**：
  - 敏感信息加密传输（HTTPS）
  - 防止 SQL 注入
  - 防止 XSS 攻击
  - 防止 CSRF 攻击
- **权限控制**：
  - 基于角色的权限管理（RBAC）
  - 接口级别权限控制
  - 防止越权操作
- **内容安全**：
  - 敏感词过滤
  - 图片审核（鉴黄、鉴暴）
  - 用户举报机制

### 4.3 易用性需求

- **界面设计**：
  - 采用现代简约风格
  - 以中国风元素为特色
  - 色彩搭配符合文化调性
- **交互体验**：
  - 操作流程简洁直观
  - 减少用户操作步骤
  - 提供清晰的操作反馈
- **响应式设计**：
  - 支持 PC 端访问
  - 支持移动端访问
  - 自适应不同屏幕尺寸
- **错误提示**：
  - 提供友好、清晰的错误提示
  - 引导用户正确操作

### 4.4 可维护性需求

- **代码规范**：
  - 遵循统一的代码规范
  - 使用设计模式提高代码质量
- **模块化设计**：
  - 采用分层架构
  - 模块职责清晰
  - 便于扩展和维护
- **文档完善**：
  - 提供详细的 API 文档
  - 提供数据库设计文档
  - 提供部署文档
- **日志记录**：
  - 记录关键操作日志
  - 记录异常日志
  - 便于问题排查

### 4.5 可扩展性需求

- **水平扩展**：
  - 支持集群部署
  - 支持负载均衡
- **功能扩展**：
  - 预留扩展接口
  - 支持插件化开发
- **数据扩展**：
  - 支持数据库分库分表
  - 支持缓存扩展

## 8. 安全设计

### 8.1 认证与授权

- **JWT Token 认证**：
  - 用户登录后颁发 JWT Token
  - Token 有效期 7 天
  - 支持 Refresh Token 刷新
  - Token 存储在 HTTP Only Cookie 或 LocalStorage

- **权限控制**：
  - 基于角色的访问控制（RBAC）
  - 接口级别权限注解
  - 前端路由守卫

### 8.2 数据安全

- **密码安全**：
  - BCrypt 加密存储
  - 密码强度校验
  - 登录失败次数限制

- **数据传输**：
  - HTTPS 加密传输
  - 敏感数据加密（手机号、身份证）

- **数据安全**：
  - SQL 注入防护（MyBatis 预编译）
  - XSS 攻击防护（输入过滤）
  - CSRF 攻击防护（Token 验证）

### 8.3 内容安全

- **敏感词过滤**：
  - 建立敏感词库
  - 实时过滤用户输入
  - 支持动态更新词库

- **图片审核**：
  - 接入第三方图片审核 API
  - 鉴黄、鉴暴、鉴政
  - 人工复审机制

- **用户举报**：
  - 支持用户举报违规内容
  - 管理员及时处理
  - 记录处理结果

## 13. 风险评估

### 13.1 技术风险

- **风险**：Spring Boot 和 Vue 的版本兼容性问题
- **影响**：中等
- **应对措施**：
  - 选择稳定版本
  - 进行充分的兼容性测试
  - 建立问题反馈机制

### 13.2 安全风险

- **风险**：用户信息和交易数据泄露
- **影响**：高
- **应对措施**：
  - 加强安全设计
  - 定期进行安全测试
  - 建立安全响应机制

### 13.3 性能风险

- **风险**：系统并发能力不足
- **影响**：中等
- **应对措施**：
  - 进行性能优化
  - 使用缓存和异步处理
  - 预留扩展空间

### 13.4 内容风险

- **风险**：文化内容准确性不足
- **影响**：中等
- **应对措施**：
  - 建立内容审核机制
  - 邀请文化专家顾问
  - 建立用户纠错机制

### 13.5 运营风险

- **风险**：卖家和买家数量不足
- **影响**：高
- **应对措施**：
  - 制定运营推广计划
  - 与文创机构合作
  - 举办文化活动

## 14. 结论

"豫见好物"河南文创产品销售平台是一个集文化展示、商品销售、文化互动于一体的综合性电商平台。通过本平台的建设，可以有效解决河南文创产品销售渠道有限、文化传播不足的问题，推动河南传统文化的创新性发展和商业价值转化。

平台采用 Spring Boot + Vue 的技术架构，具有良好的可扩展性、易维护性和高性能。通过合理的功能模块划分、数据库设计和技术选型，确保系统能够满足业务需求并具备良好的用户体验。

同时，平台注重文化特色的体现，通过文化标签、文化背景、工艺说明等功能，让用户在购物的同时深入了解河南文化，实现"以文促商、以商传文"的良性循环。
